# 🍅 番茄钟小程序

一个基于微信小程序的番茄工作法计时工具，帮助提高专注力和工作效率。

## ✨ 功能特性

- ⏱️ **灵活计时**：支持 1-60 分钟自定义专注时长
- 📊 **任务分类**：6 种任务类型（工作、学习、思考、写作、运动、阅读）
- 🎨 **可视化进度**：Canvas 绘制的圆形进度条，直观显示剩余时间
- ⏸️ **暂停/继续**：支持暂停和继续功能，灵活控制专注时间
- 📈 **数据统计**：记录每次专注数据，支持今日和历史统计
- 💾 **本地存储**：数据保存在本地，无需网络连接

## 🛠️ 技术栈

- **微信小程序** + **JavaScript** + **WXML** + **WXSS**

## 🚀 快速开始

### 环境要求

- 微信开发者工具（最新版本）
- 微信小程序 AppID（可在 [微信公众平台](https://mp.weixin.qq.com/) 注册获取）

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/RK332/tomatoClock.git
   cd tomatoClock
   ```

2. **配置 AppID**
   - 打开 `project.config.json`
   - 将 `appid` 字段替换为你的小程序 AppID

3. **打开项目**
   - 使用微信开发者工具打开项目目录
   - 选择"小程序"项目类型

4. **运行项目**
   - 点击"编译"按钮即可在模拟器中预览
   - 点击"预览"按钮生成二维码，用微信扫码在手机上体验

## 📖 使用说明

### 开始专注

1. 在首页使用滑块选择专注时长（1-60 分钟）
2. 选择一个任务类型（工作/学习/思考/写作/运动/阅读）
3. 点击"开始专注"按钮
4. 专注过程中可以暂停或放弃

### 查看统计

1. 切换到"统计"标签页
2. 查看今日和累计的专注数据
3. 切换"今日"和"历史"查看不同时间段的记录

## 📝 项目配置

### project.config.json

```json
{
  "appid": "你的小程序AppID",
  "projectname": "tomatoClock",
  "miniprogramRoot": ""
}
```
